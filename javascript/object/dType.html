객체가 아닌 데이터 타입

문자열은 프로퍼티와 메소드가 존재하기 떄문에 객체이지만 왜 객체라고 하지 않을까?

문자열은 원시 데이터 타입이고 문자열과 관련된 어떤 작업을 하려 할 때 자바스크립트는 임시로 문자열 객체를 만들고 제거한다.

이러한 처리는 내부적으로 일어난다.
