숫자의 표현

  • MATH.pow(3,2) 3의 2승
  • MATH.round(10.6) 10.6을 반올림
  • MATH.ceil(10.2) 10.2를 올림
  • MATH.floor(10.6) 10.6을 내림
  • MATH.sqrt(9) 3의 제곱근
  • MATH.random(); 0부터 1.0사이의 임의의 숫자

문자의 표현

  • '와 "를 이용하여 문자표현 가능
  • \'나 \"를 이용해 '나 "를 표현가능
  • "1"은 String type
  • +연산자를 통해 문자와 문자를 결합가능
  • .length를 통해 길이를 구할 수 있다.
  • .indexOf()를 통해 해당 문자가 몇번자리에 존재하는지 확인가능
