inline vs block level

레이아웃1 레이아웃2

html 태그는 화면전체를 사용하는 태그와 자기 자신의 크기만큼만을 사용하는 태그로 나누어진다.

왜 h2 태그는 줄바꿈이 되는가?

테두리를 보면 h2는 줄 전체를 사용하고 a는 자신의 자리만 사용하고 있다.

자신과 자신을 둘러싸고 있는 텍스트와 하나의 같은 줄에 위치하는 형태의 태그를 inline element

자신 혼자 화면전체를 다 쓰는 태그를 block level element라고 한다.

display: 를 통해 inline과 block level을 설정 해 줄 수 있다.
