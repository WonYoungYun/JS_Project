Flex에대해 더 알아보기

레이아웃을 위한 기능

flux를 위한 container와 item

    container

  • display = flex설정을 해줌
  • flex-direction = 설정을통해 정렬위치변경 (기본값은 row)
  • flex-wrap
  • flex-flow
  • justify-content = flex로 설정한 레이아웃의 브라우저 상의 수평 위치를 바꾼다.
  • align-items = flex로 설정한 레이아웃의 브라우저상의 수직 위치를 바꾼다
  • align-content = align-items과 비슷하지만 items는 container안의 전체 element를 하나의 그룹으로보고
    align-content는 같은 행을 하나의 그룹으로 쳐서 그룹과 그룹사이의 정렬을 결정해줌

    item

  • order = element에 rank를 부여하여 출력순서를 바꿀 수 있다.
  • flex-grow = container의 item이 container의 여백을 채움(기본값0)
  • flex-shrink = 브라우저의 창을 줄였을때 element의 크기를 shirnk/전체shrink값 만큼 분담함
    (0이면 작아지지 않는다, 기본으로 basis값을 가지고 있어야 한다.)
  • flex-basis = element의 기본 크기 지정
  • flex = flex-grow, flow-shrink, flow-basis를 축약
  • align-self = align-items로 지정된 상태에서 특정한 것만 이동시키기 위해서 align-self를 사용
