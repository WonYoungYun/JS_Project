Font-Family

family

font family를 통해 보여주고싶은 폰트를 우선순위로 정할 수 있다. 이 때 마지막 폰트를 포괄적인 폰트(serif, monospace 등)로 지정해야 한다.

monospace는 고정폭을 의미하다며, 가변폭은 자연스럽지만 고정폭은 데이터를 다룰때 일목요연한 형태로 정리가 가능

Font-wieght

weight

weight의 설정을 통해 텍스트를 굵게 표시가능하다

Line-height

Line Height
Line Height

기본값이 1.2

px과 같은 고정된 값은 피하고 2(2배) 같은 상대적인 수치를 사용하는게 좋다.

font:bold 5rem/2 arial, verdana, "Helvetica Neue", serif 로 축약 표현이 가능하다
